🧪 QM/ML Molecular Dynamics Project

This repository contains input files and scripts for performing QM/ML simulations.

📁 File Descriptions
File Name	       | Description

base.cp2k.in     | Template input file for CP2K (DFT/Ab-initio calculations).

data.cp2k	       | Structural/Coordinate data for CP2K simulations.

base.in.lammps	 | Configuration file for LAMMPS molecular dynamics runs.

data.lammps	     | Topology and initial coordinate data for LAMMPS.

qmml.py	         | Python script for the Quantum Mechanics/Machine Learning interface.

run.slm	         | Slurm file for high-performance computing (HPC).
